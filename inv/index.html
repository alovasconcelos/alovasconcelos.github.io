  • ASCII Invaders

    Esta é uma versão simplificada (e muito!) do clássico jogo Space Invaders, usando caracteres ASCII e rodando em console. Desenvolvido em C# usando o Visual Studio Community, apesar de bastante modesto, este jogo apresenta dois recursos interessantes para uma aplicação do tipo console, a saber: o uso de cores e efeitos sonoros. O objetivo deste projeto é meramente didático, apresentando aspectos bem básicos da programação de jogos.
